## 司机端
---
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;基于微信小程序和腾讯地图的网约车项目。  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;未采用第三方组件库，部分采用微信小程序原生组件库[weui](https://weui.io/)，UI100%还原蓝湖设计稿。

## 说明，项目目前由于微信小程序地图SDK缺乏导航功能，故此转向原生APP开发，此项目开源。
